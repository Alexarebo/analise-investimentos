# ANÁLISE DE INVESTIMENTOS
Planilha de análise de investimentos


## **OBJETIVO**

  O  objetivo é criar uma planilha para analisar aportes em determinado investimento criando cenários futuros conforme os anos. No caso os tipos de fundos de investimento imobiliário.


## **CONCEITO**

FIIs (Fundos de Investimento Imobiliário) são veículos coletivos para investir no mercado imobiliário sem comprar imóveis diretamente, isto é, não precisa comprar uma casa completa ou um gapão completo. Pode comprar parte deste empreendimento reunindo dinheiro de vários investidores para aplicar em imóveis físicos (shoppings, galpões), títulos imobiliários (CRIs, LCIs) ou cotas de outros FIIs, gerando rendimentos (aluguéis/juros) e valorização para os cotistas, tudo negociado na Bolsa de Valores. 

## **DADOS BÁSICOS DE ANÁLISE**

Pode-se definir a base do salário que deve ser investido por uma pessoa por mês. Vai depender de quanto recebe. Entretanto, via de regra vamos utilizar 30% do salário.
